Design of a Game Controller

2024 Hardware Design

Designed and built a custom game controller from scratch, focusing on ergonomics, functionality, and user experience. This project combined hardware design, electronics, and programming to create a fully functional gaming peripheral.

Key Features

  • Ergonomic button layout optimized for comfort during extended gaming sessions
  • Programmable buttons with customizable key mapping
  • Custom PCB design with microcontroller integration
  • USB connectivity for plug-and-play functionality
  • Tactile feedback system for enhanced user experience

Technical Implementation

The controller features a custom-designed PCB housing an Arduino-compatible microcontroller. Button inputs are processed through debouncing circuits to ensure accurate registration. The firmware allows users to remap buttons through a simple interface, making it adaptable to different gaming preferences.

Communication Between ESP32 Microcontrollers

2024 IoT & Wireless

Developed a robust wireless communication system between two ESP32 microcontrollers using multiple protocols. This project explores various communication methods including WiFi, Bluetooth, and ESP-NOW for different use cases in IoT applications.

Communication Protocols Implemented

  • WiFi Communication: Established TCP/IP socket connection for reliable data transfer
  • Bluetooth Classic: Serial communication for legacy device compatibility
  • Bluetooth Low Energy (BLE): Energy-efficient communication for sensor data
  • ESP-NOW: Fast, connectionless communication protocol for real-time applications

Applications & Use Cases

The system demonstrates practical applications such as remote sensor monitoring, distributed control systems, and mesh networking. Data encryption ensures secure communication between devices, making it suitable for sensitive IoT deployments.

Performance Metrics

  • Range: Up to 100 meters in open space (WiFi/ESP-NOW)
  • Latency: Under 10ms with ESP-NOW protocol
  • Data throughput: Up to 1 Mbps with optimized packet structure
  • Power consumption: Optimized deep sleep modes for battery operation

PyQt Application interacting with OS

2025

Developed a desktop application using the PyQt framework to open several .exe applications on the screens detected by the OS.
The application's code is hosted on GitHub .

How it works:
The user selects all .exe applications, which are then added to the scroll view on the left of the application.
The user can then assign these applications to the screens detected by the OS.
Finally, clicking "Launch applications" will open them on the selected screens.

  • PyQt5 framework for building the GUI
  • Interaction with the OS using libraries (os, win32api)
  • Automated task scheduling and execution

Website Design & Development

2025

Led an AI agent directly integrated into the VS Code IDE to develop this portfolio website.

This project allowed me to discover the advantages and disadvantages of exclusively using an AI agent for a coding project.

Conclusion on Exclusively Using an AI Agent

✓ Advantages

  • Very fast for prototyping small ideas
  • Automates repetitive tasks and small changes
  • Learned the strengths and weaknesses of several AI agents

✗ Disadvantages

  • Learned very little about the subject
  • Difficult to modify and read the generated code structure
  • Need to limit its scope and provide good prompts to prevent bugs
  • Difficulty implementing highly customized or unique features

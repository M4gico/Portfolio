Serma Safety & Security

SERMA Safety & Security supports companies in ensuring the safety and operational security of their electronic products.
I work within the security laboratory, which evaluates complex secure products ranging from electronic chips to complete hardware systems.

Cybersecurity Hardware Evaluation

Mission

Participate on the hardware field in the conformity evaluation of a secure product. The evaluation is made compared of security specification of the client and under the control of a certification body.

Improvements

  • Semi-conductor physics and microcontroller architectures
  • Hardware security evaluation techniques (Laser, EM, BBI, Glitch)
  • Signal analysis and fault injection methodologies
  • Working with other experts of different fields (software, cryptography)
Mission Example

On a smart card product that will be used for banking transaction on the MasterCard scheme, I was in charge to trying to bypass the PIN code verification using laser fault injection on the backside of the chip.
To do so, with the help of a software expert, I made a signal analysis of the current consumption to see the difference between a correct PIN and an incorrect PIN verification.
After identifying the right time frame, I proceeded to the laser fault injection campaign to try to bypass the PIN code verification.

R&D - Emission Microscopy

Mission

Developed an HCI (Human Control Interface) using PyQt library for controlling translation platine and near infrared camera to map the emission of a chip.
Additionally, added integrated picture treatment (subtraction, averaging, overlay) directly in the GUI.

Improvements

  • Object-Oriented Programming (OOP) with Python and PyQt framework
  • Emission Microscopy techniques and chip emission analysis
  • Semi-conductor physics and microcontroller architectures
  • Hardware Control Interface (HCI) design and development
Project Development Process

I retrieved a start of the GUI made by a previous intern. This GUI had translation stage communication and showed the camera feed.
Functionality I added:

  • Camera parameters adjustment (exposure, gain, temperature)
  • Control speed and acceleration of the translation stage
  • Subtract, average and overlay pictures
  • Can add picture treatment directly in the GUI
  • Map all chip with 3 corner points

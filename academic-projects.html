ACADEMIC PROJECTS

Sumo Robot

Sep to Dec 2024

Project Objective

Designed and verified the sensor integration of a tracker robot for competitive sumo robot matches. The goal is to push the opponent robot out of the ring using various sensors and control algorithms.

Project Development Process

The main part of this project was design and choose which and how many sensors to use for the better detection of an opponent robot.
For this, we created a Design file to explain our choices and the working principles of each sensor. After the design phase, we created prototypes to test our designs and program in Arduino the converter of the sensors signals to a measure unit value.
In the same time, I also helped the treatment team to program the algorithm of the robot.
After validating the prototypes, we created in our university lab the PCB and soldered all CMS components.
Finally, we integrated the sensors into the sumo robot PCB and verified their functionality in real match conditions.

All choices and verifications were written in report documenting.

Improvements

  • Better understanding on sensors integration
  • Working with a team (8 peoples)
  • PCB designs and soldering CMS components
  • Low level programming for sensor integration and algorithms

Propeller Kart

Jan to Jun 2024

Project Objective

Designed and verified the action part of the propeller-driven kart and the sensor part of the accompanying infrared (IR) remote control. The two systems communicate using the NEC protocol to communicate wirelessly.

Project Development Process

My role in this project was focused on both the kart's hardware and the remote's software. I was responsible for the "action" part of the kart, which included integrating the propeller motor and the buzzer.
I also handled the "sensor" part, which involved implementing the IR communication receiver.
A significant part of my contribution was developing the entire Arduino code for the remote control's microprocessor.

Improvements

  • Understanding of IR communication and the NEC protocol
  • Low level development for a microprocessor (Arduino)
  • Integrating hardware components with a custom-coded remote control

Propeller kart with IR control system and PCB

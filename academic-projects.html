ACADEMIC PROJECTS

Sumo Robot

Third Semester

Designed and verified the sensor integration of a tracker robot for competitive sumo robot matches. This project involved comprehensive work on embedded systems, sensor fusion, and PCB design.

The sumo robot project challenged us to create an autonomous robot capable of detecting opponents and pushing them out of a ring, requiring precise sensor integration and real-time decision-making algorithms.

Key Contributions

  • Designed and integrated multiple sensors for opponent detection and edge detection
  • Verified sensor accuracy and calibration for optimal performance
  • Contributed to PCB design and layout for compact robot integration
  • Implemented sensor fusion algorithms for improved tracking

Technologies Used

  • Embedded C programming for microcontroller control
  • IR sensors for opponent detection
  • Line sensors for ring edge detection
  • Motor control systems for movement
  • PCB design software for circuit layout

Propeller Kart

Second Semester

Designed and verified the data acquisition system for a propeller-driven kart, combined with an infrared (IR) remote control system. This innovative project explored alternative propulsion methods and wireless control systems.

The propeller kart project required careful integration of power electronics, control systems, and wireless communication to create a functional and safe vehicle controlled remotely via IR signals.

Project Highlights

  • Designed data acquisition system for monitoring kart performance
  • Implemented IR remote control for wireless operation
  • Contributed to PCB design for power distribution and control
  • Verified system safety and performance under various conditions
  • Integrated propeller motor control with remote commands

Technical Implementation

  • IR receiver and transmitter circuits for remote control
  • Motor speed control using PWM signals
  • Data acquisition sensors for speed and performance monitoring
  • Power management circuitry for safe operation
  • PCB design and implementation for compact integration
